### Technologies

- React App
- [Deck.gl](https://deck.gl/docs/get-started/using-with-react)
- [Mapbox](https://www.mapbox.com/)

### Frontend (App)

- `yarn install`
- insert MAPBOX_ACCESS_TOKEN in src/App.js
- `yarn start`
- localhost:3000

### Backend

- `npm run start`
- localhost:9000
